# 그리디 알고리즘

## 뜻

- 현재 상황에서 지금 당장 좋은 것만을 고르는 방법
- 문제를 풀기 위한 최소한의 아이디어를 떠올릴 수  있는 능력 → 최적의 해를 구할 수  있는가에 대한 검토 필수적
- 우선 당장 좋은 것만 고르는 건데, 그 방식대로 하는 것이 최적임을 추론할 수 있어야 문제가 풀림.
  예시) 왜 당장 좋은 것만 고르는 것이 최적인지에 대한 이유가 있어야 함.
